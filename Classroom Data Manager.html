How goal ID matching works

Students store goal references as IDs (G001, G032, G063…) — never the goal text. So when you update a goal description in your Sheet, all student links stay intact automatically.

A warning only appears if you delete a goal row entirely from the IEP Goals tab. The integrity checker flags which students are affected and which ID is missing, so you can decide: restore the goal row, clean up the student record, or replace it with a new ID.
